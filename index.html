¿Cómo está organizado?

Sobre CoreSystem v2.0

CoreSystem es un ecosistema frontend modular construido únicamente con estándares web: HTML5, CSS3 y Vanilla JavaScript ES6+.

CSS — Arquitectura ITCSS

  • Variables & Design Tokens
  • Reset y estilos base
  • Sistema de layout (Grid + Flex)
  • 30+ componentes UI
  • Utilidades atómicas
  • Animaciones y scroll reveal

JS — Patrón IIFE

  • 14+ módulos independientes
  • Sistema de validación de formularios
  • Scroll reveal con IntersectionObserver
  • Lightbox para galerías
  • Dark mode con persistencia
  • API pública: CoreSystem.*
Sin dependencias. No necesitas Node.js, npm, webpack ni ningún build tool. Simplemente vincula styles.css y script.js.
